# postgraduate-process-server
基于SpringBoot+Vue前后端分离的考研流程监控管理系统，系统包含管理员、学生角色，功能如下： 管理员：首页统计（录取统计、教师预约统计）；账号管理；报名登记；准考证书；志愿管理；初试分数；复试成绩；教室信息；座位申请；留言信息；公告信息；个人信息；密码修改。 用户：注册、登录系统；报名登记（提交登记，确认报名）；准考证书；志愿管理；初试分数；复试成绩；教室信息（查看座位）；座位申请；留言信息；个人资料、修改密码。 系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，有需要可提供指导。
